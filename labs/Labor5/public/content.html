HTML5

HTML5 ist die neueste Entwicklung der Hypertext Markup Language, der Grundstruktur für alle Webseiten.

CSS ist eine Stylesheet-Sprache, die das Erscheinungsbild von HTML-Dokumenten beschreibt.

JavaScript ist eine leistungsfähige Programmiersprache, die auf Bögen und Webservern ausgeführt wird.

FlexBox ist ein Layout-Modell, das eine effiziente Anordnung von Elementen innerhalb eines Containers ermöglicht.

GridBox ist ein Layout-System, das komplexe zweidimensionale Grid-basierte Layouts ermöglicht.
